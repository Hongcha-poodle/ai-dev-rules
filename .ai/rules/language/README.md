# Language-Specific Rules

이 디렉토리에는 언어/프레임워크별 전문 규칙이 위치합니다.
CLAUDE.core.md §4에서 참조됩니다.

## 파일 명명 규칙

```
.ai/rules/language/
├── go.md          # Go 언어 규칙 (테스트 명령어, 포매팅 등)
├── python.md      # Python 규칙
├── typescript.md  # TypeScript/JavaScript 규칙
└── ...
```

각 파일에는 해당 언어의 다음 항목을 포함합니다:
- 테스트 실행 명령어
- 포매팅 도구 및 설정
- 린터 설정
- 빌드 명령어
- 언어별 주의사항
